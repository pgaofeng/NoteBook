# NoteBook

## 使用说明

1. 安装node.js
2. npm install hexo-cli
3. hexo init blog
4. rm -rf blog/source
5. mv source blog
6. cd blog
7. mv source/_post/img source
8. npm install hexo-renderer-pug --save
9. npm install hexo-renderer-sass --save
10. npm install hexo-simple-mindmap --save
11. hexo g
12. hexo s
